¿Qué es Metasoc?
Metasoc nace de la unión de dos grandes tradiciones humanas: la filosofía griega (μετά — "más allá", "trascender") y la disciplina militar romana (el arte de la conquista organizada). Somos una plataforma para conquistar metas personales y empresariales aplicando la misma metodología que llevó a Roma a dominar el mundo conocido.

No se trata de motivación vacía. Se trata de estrategia, disciplina y filosofía aplicada. Cada meta es una campaña. Cada obstáculo, un enemigo a estudiar. Cada victoria, una lección para la siguiente batalla.

Infraestructura del Imperio
El Imperio Metasoc opera con una filosofía de máxima eficiencia y mínimo costo — como la Legión Romana que conquistó el mundo con disciplina más que con riqueza:

🏰 Frontend: HTML + CSS + JavaScript puro — sin frameworks innecesarios
🗺 Deploy: Cloudflare Pages — sitio estático, deploy automático desde GitHub
🤖 IA Local: Ollama + OpenClaw — modelos locales para generar reportes sin costo
📊 Datos: JSON plano — simple, versionable, editable por humanos y máquinas
🔄 Automatización: Cron job — deploy cada hora sin intervención manual
💰 Costo mensual: < $15 USD — el Imperio romano nunca fue costoso en lo esencial
